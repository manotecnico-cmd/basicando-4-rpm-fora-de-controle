# basicando-4-rpm-fora-de-controle
Estudo de caso técnico interativo sobre sobre-rotação em grupo gerador Cummins NTA855G (~500 kVA) com comando PowerCommand (PCC3100). Do sintoma ao diagnóstico real, mostrando como o básico bem feito (regulagem mecânica) resolveu o que parecia eletrônico ou hidráulico. Projeto educativo da Infinita Geradores.
